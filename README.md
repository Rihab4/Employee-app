Employee App

Employee App is a web application used to manage employee records.

*Features

The application allows you to:

 -View all employees

 -View employee details
 -Add a new employee

 -Delete an employee

*Deployment & Architecture

This application is deployed on Amazon Web Services (AWS) and includes the following components:

 -EC2 – hosts the Spring Boot application

 -Amazon Aurora MySQL – used as the relational database

 -Amazon S3 – used for storing employee photos
